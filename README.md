# README #

A slackbot that uses JMegaHal (http://www.jibble.org/jmegahal/) to learn language structure

## Start Instructions ##
scalaVersion := "2.11.6", sbt.version = 0.13.9

Compile:

```
sbt compile
```


to Run: 

```
sbt clean run
```
